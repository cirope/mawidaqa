Organizar documentos

Los documentos se listan ordenados por su código en órden alfabético, pero se pueden organizar de una manera que tenga sentido para usted

Etiquetas

Puede etiquetar documentos para agruparlos de manera lógica, puede utilizar tantas etiquetas como considere necesario

Ejemplo Si tiene un manual de un procedimiento, podría etiquetarlo como:

  • Manuales
  • Procedimientos
